# ચમકદાર (Chamakdar) - High-Converting Facebook Ad Landing Page

একটি প্রিমিয়াম, দ্রুত লোড হওয়া এবং মোবাইল-অপ্টিমাইজড ল্যান্ডিং পেজ যা আপনার ব্যবসা **চমকদার**-এর মিনি সেলাই মেশিন এবং ফুড গ্রাইন্ডার বিক্রির জন্য ডিজাইন করা হয়েছে।

## 📁 ফাইল স্ট্রাকচার

- `index.html`: মূল ল্যান্ডিং পেজের এইচটিএমএল কোড। এটি এসইও ফ্রেন্ডলি ও আকর্ষণীয় সেকশনে সাজানো।
- `style.css`: ল্যান্ডিং পেজের চোখ ধাঁধানো কালার স্কিম, অ্যানিমেশন, ও রেসপনসিভ লেআউট স্টাইল।
- `app.js`: অটো-প্রোডাক্ট সিলেকশন, লাইভ প্রাইস ক্যালকুলেশন এবং গুগল ফর্ম সাবমিশন লজিক।
- `config.js`: সহজে ফোন নাম্বার, প্রোডাক্টের দাম, ডেলিভারি চার্জ পরিবর্তন করার কনফিগারেশন ফাইল।
- `assets/`: প্রোডাক্টের প্রিমিয়াম ছবি ফোল্ডার।

## 🚀 কিভাবে প্রজেক্টটি চালু করবেন (How to Run)

এই ল্যান্ডিং পেজটি সম্পূর্ণ ভ্যানিলা এইচটিএমএল, সিএসএস ও জাভাস্ক্রিপ্ট দিয়ে তৈরি করা হয়েছে, তাই কোনো সার্ভার ইন্সটল করার ঝামেলা ছাড়াই এটি চালু করা যায়:
1. `index.html` ফাইলটিতে ডাবল-ক্লিক করে যেকোনো ব্রাউজার (Chrome, Firefox, Safari) এ ওপেন করুন।
2. অথবা ডেভলপমেন্টের সুবিধার জন্য VS Code-এর **Live Server** এক্সটেনশন ব্যবহার করতে পারেন।

## 📝 গুগল ফর্ম ইন্টিগ্রেশন গাইড (Google Form Setup Guide)

ফ্রি এবং কম খরচে কাস্টমারদের অর্ডার ডাটা সংগ্রহ করার জন্য এই ফর্মটি ব্যাকগ্রাউন্ডে গুগল ফর্মের সাথে লিঙ্ক করা যায়। ইন্টিগ্রেশন করতে নিচের সহজ ধাপগুলো অনুসরণ করুন:

### ধাপ ১: গুগল ফর্ম তৈরি করুন
1. [Google Forms](https://docs.google.com/forms/)-এ যান এবং একটি নতুন ফর্ম তৈরি করুন।
2. ফর্মে নিচের প্রশ্নগুলো যোগ করুন (সবগুলো **Short Answer** অথবা **Paragraph** সিলেক্ট করুন):
   - কাস্টমারের নাম
   - মোবাইল নম্বর
   - ডেলিভারির সম্পূর্ণ ঠিকানা
   - সিলেক্টেড প্রোডাক্ট
   - পরিমাণ (Quantity)
   - ডেলিভারি এরিয়া
   - সর্বমোট বিল (Total Price)

### ধাপ ২: ফর্মের Action URL বের করুন
1. ফর্মের উপরে ডান কোণায় **Send** বাটনে ক্লিক করুন এবং লিঙ্ক চিহ্নে ক্লিক করে ফর্মের লিঙ্কটি কপি করুন।
2. ব্রাউজারে একটি নতুন ট্যাবে লিঙ্কটি ওপেন করুন।
3. ফর্ম পেজের যেকোনো খালি জায়গায় রাইট ক্লিক করে **Inspect** অথবা **View Page Source**-এ যান।
4. `Ctrl + F` টিপে সার্চ করুন: `<form`
5. সেখানে `action="https://docs.google.com/forms/d/e/XXXXXXXXXX/formResponse"` লেখা লিঙ্কটি পাবেন। এই পুরো লিঙ্কটি কপি করুন। এটিই আপনার `actionUrl`।

### ধাপ ৩: ফিল্ডের Entry IDs (যেমন entry.123456) বের করুন
1. আপনার গুগল ফর্মের লিঙ্কে গিয়ে কাস্টমার সেজে একটি ডেমো ফর্ম পূরণ করুন (সাবমিট করবেন না)।
2. রাইট ক্লিক করে **Inspect** (পরীক্ষণ) সিলেক্ট করুন।
3. ফর্মের প্রতিটি ইনপুট বক্সের `name` অ্যাট্রিবিউট চেক করুন। যেমন: `<input type="text" name="entry.18475269">`।
4. প্রতিটি ফিল্ডের (নাম, মোবাইল নম্বর, ইত্যাদি) পাশের এই `entry.XXXXXXXX` সংখ্যাগুলো লিখে রাখুন।

### ধাপ ৪: config.js ফাইলে বসান
আপনার `config.js` ফাইলটি খুলুন এবং পরিবর্তন করুন:
```javascript
googleForm: {
  enabled: true, // এটি true করে দিন
  actionUrl: "এখানে আপনার কপি করা action URL টি বসান",
  entryIds: {
    name: "entry.আপনার_নাম_ফিল্ডের_আইডি",
    phone: "entry.আপনার_মোবাইল_ফিল্ডের_আইডি",
    address: "entry.আপনার_ঠিকানা_ফিল্ডের_আইডি",
    product: "entry.আপনার_প্রোডাক্ট_ফিল্ডের_আইডি",
    quantity: "entry.আপনার_পরিমাণ_ফিল্ডের_আইডি",
    delivery: "entry.আপনার_ডেলিভারি_ফিল্ডের_আইডি",
    totalPrice: "entry.আপনার_সর্বমোট_বিল_ফিল্ডের_আইডি"
  }
}
```

এটি করার পর, কোনো কাস্টমার যখন ওয়েবসাইটে অর্ডার ফর্ম সাবমিট করবেন, তার নাম-ঠিকানা সরাসরি আপনার গুগল শিট বা গুগল ফর্মে চলে যাবে এবং কাস্টমার ওয়েবসাইটে একটি প্রিমিয়াম সাকসেস মেসেজ দেখতে পাবেন।

## 📞 কাস্টমাইজেশন (Customization)

ফোনের নাম্বার বা ফেসবুক লিঙ্ক পরিবর্তন করতে `config.js` ফাইলটি খুলুন এবং নিচের অংশ পরিবর্তন করুন:
- `contactNumber`: "আপনার মোবাইল নাম্বার দিন"
- `whatsappNumber`: "আপনার হোয়াটসঅ্যাপ নাম্বার দিন"
- `facebookPageUrl`: "আপনার ফেসবুক পেজ লিংক"
- `deliveryCharges`: ঢাকা সিটির ভিতরে ও বাইরে ডেলিভারি চার্জ নির্ধারণ করুন।
